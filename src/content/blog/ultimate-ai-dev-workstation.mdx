## 那个让我崩溃的凌晨

凌晨 2:47，生产环境报 Bug。

我同时开着：
- 3 个 VS Code 窗口（前端、后端、配置文件）
- 2 个 PowerShell 窗口（跑服务、看日志）
- 1 个 Git Bash（拉代码）
- 1 个浏览器（查文档）
- 1 个 Claude AI（问问题）

**Alt + Tab 切得我手指酸，眼神开始涣散。**

最崩溃的是——我想问 AI 一个问题，但之前的对话记录在另一个窗口，我找不到了。

那一刻我意识到：**这不是"工具"问题，这是"操作系统"问题。**

于是我决定：不再忍受，我要造一个真正属于开发者的"开发操作系统"。

经过 6 个月的迭代，**AI Dev Workstation** 诞生了。

## 一分钟看懂：它是什么？

想象一下，一个命令 `of start`：

✨ 你的编辑器自动打开，定位到当前项目
✨ 终端自动分成 4 个面板：
- 左上：Claude AI（架构设计）
- 右上：Codex AI（代码生成）
- 左下：Gemini AI（文档查询）
- 右下：Shell（git、服务、工具）

**一个屏幕，4 个 AI，同时为你工作。**

## 为什么说它是"革命性"的？

### 1️⃣ 多项目管理，秒级切换

之前：每次切项目，都要重新打开编辑器、重新定位目录
现在：`of use myproject`，一个命令，环境就绪

```bash
# 注册项目（在项目目录下）
of register myproject

# 切换项目
of use myproject

# 开始工作（编辑器 + AI 全部就位）
of start
```

### 2️⃣ AI 多开，协作编程

之前：开一个 AI，问完关掉，再开另一个
现在：4 个 AI 同时在线，各司其职

- **Claude**：负责架构和复杂推理
- **Codex**：负责代码片段生成
- **Gemini**：负责文档和算法查询
- **Shell**：负责 git、日志、工具操作

### 3️⃣ JSON 配置，跨平台同步

之前：Bash 脚本配置，换机器要重新配置
现在：JSON 配置文件，Git 同步，到处可用

```json
{
  "current": "myproject",
  "default_editor": "antigravity",
  "editors": {
    "antigravity": "C:\\...\\Antigravity.exe",
    "vscode": "C:\\...\\Code.exe"
  },
  "projects": {
    "myproject": {
      "win_path": "F:\\Git\\myproject",
      "wsl_path": "/mnt/f/Git/myproject",
      "editor": "antigravity"
    }
  }
}
```

### 4️⃣ 一键安装，开箱即用

```bash
git clone https://github.com/ai-dev-workstation/ai-dev-workstation.git
cd ai-dev-workstation
./install.sh
```

5 分钟后，你的"开发操作系统"就绪。

## 技术栈：为什么是这 4 件套？

| 组件 | 作用 | 为什么选它 |
|------|------|-----------|
| **WSL2** | Linux 内核 | 原生性能，不是虚拟机 |
| **WezTerm** | 终端模拟器 | GPU 渲染，比 Windows Terminal 更丝滑 |
| **Zellij** | 多窗管理 | 比 tmux 更现代，自带快捷键提示 |
| **of** | 项目管理器 | 核心工具，本文主角 |

## 真实场景：它改变了什么？

### 场景 1：紧急排查 Bug

**之前**：
1. 打开日志窗口
2. 打开代码编辑器
3. 打开 AI 问问题
4. 来回切换，迷失在窗口海洋

**现在**：
```bash
of start
```
一个命令，所有窗口就位，左右分屏，一目了然。

### 场景 2：多项目并行

**之前**：
- 项目 A 需要前端、后端两个窗口
- 项目 B 又要两个窗口
- 桌面上 6-7 个窗口，不知道哪个是哪个

**现在**：
```bash
of use project-a
of start
```
每个项目独立环境，秒级切换。

### 场景 3：AI 协作编程

**之前**：
- 问 Claude 架构，关掉
- 问 Codex 代码，关掉
- 问 Gemini 文档，关掉
- 对话记录全丢失

**现在**：
4 个 AI 同时在线，对话记录永久保存，随时回顾。
